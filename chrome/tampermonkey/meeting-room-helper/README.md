# 会议室助手 (Meeting Room Helper)

## 项目简介

这是一个 Tampermonkey 用户脚本，用于优化公司会议室预订系统的使用体验。帮助用户快速筛选出相关楼层的会议室，提高预订效率。

### 使用场景

在会议室预订页面中，系统会显示所有楼层的会议室，包括北塔（N开头）和连廊楼层。对于不需要这些楼层会议室的用户来说，这些选项会干扰视线，增加选择成本。本脚本通过一键删除这些无关楼层，让用户更快找到目标会议室。

## 功能特性

- ✅ 自动识别并删除北塔楼层（N开头）的会议室
- ✅ 自动识别并删除连廊楼层的会议室
- ✅ 提供可视化操作按钮，支持手动触发删除
- ✅ 自动适配页面动态加载，实时监听 DOM 变化
- ✅ 美观的按钮样式，与原页面风格一致

## 技术实现

### 核心技术栈

- **JavaScript (ES6+)**: 使用现代 JavaScript 语法编写
- **Tampermonkey**: 浏览器用户脚本管理器
- **DOM API**: 原生 DOM 操作，无第三方依赖
- **MutationObserver**: 监听页面动态变化
